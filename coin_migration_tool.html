🔧 Coin Migration Tool

Purane system (100 coins = ₹1) se naye system (1 coin = ₹1) mein convert karega. Sabke coins ko ÷100 karega — ek hi baar chalana hai.

⚠️ Zaroori baatein

1. Pehle "Scan" dabao — yeh sirf dikhayega kitne users affect honge, kuch change nahi karega.
2. Phir "Migrate" dabao — yeh actual coins ko Firebase mein update karega.
3. Jin users ka migration ho chuka hoga unhe dobara skip kar dega (safe to re-run).
